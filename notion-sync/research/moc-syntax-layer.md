# Syntax layer — learning order

**Thesis relevance:** LLMs must parse/generate *description language* — this domain defines what that language is.

1. Manchester Syntax 速查
2. 存在限制 (existential restriction)
3. 全称限制 (universal restriction)
4. 数量限制 (cardinality restriction)
5. 否定与不相交
6. 角色特性 (transitive, inverse, functional)
7. DL / Manchester / Functional 互转规则总表

## Serializations

- Functional-Style Syntax
